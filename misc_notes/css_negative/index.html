サイトの色を反転させる CSS Negative

2020-12-30

この About サイトは元々 npq-hugo を使っているので、
全体的にダークカラーで統一されていたんだけど、
個人的には明るい色の方がいいなあって思っていたので色を反転させるツールないかなと探したらあった。

CSS Negative

使ってみたらこのとおり、めちゃくちゃいい感じに明るい色になった！
